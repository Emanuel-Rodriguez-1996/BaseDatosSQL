Proyecto enfocado en el diseño y desarrollo de una base de datos completa, comenzando con la creación del Modelo Entidad‑Relación 
para definir entidades, atributos y relaciones. A partir de este modelo se implementó la estructura dentro del área de Bases de datos,
incluyendo la creación de tablas y relaciones, así como el desarrollo de funciones, triggers y vistas para automatizar procesos, 
optimizar consultas y mejorar la organización y gestión eficiente de la información.
